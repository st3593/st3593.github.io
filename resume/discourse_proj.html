Discourse Personal Notes Plugin

What is Discourse?

Discourse is an open source forum software founded in 2013 by Jeff Atwood (co-founder of Stack Overflow), Robin Ward, and Sam Saffron.

What does the Personal Notes plugin do?

Personal Notes is a Discourse plugin that allows users to take down notes while scrolling through forum posts and save notes for future viewing.

Front End - Languages / Concepts Used

  • HTML, SCSS, JavaScript
  • Handlebars
  • Discourse API
  • Virtual DOM, Widgets, Plugin Outlets

Key Features

1) User Customization

  • Note Color
  • Font
  • Font Style
  • Font Size

2) Convenient

  • Button located in the top-left corner below the Discourse header

3) Non-intrusive

  • Occupies minimal space
  • Button can be toggled to show/hide the personal notes interface
  • Styling options can be tucked away
